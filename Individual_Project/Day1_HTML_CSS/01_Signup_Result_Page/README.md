 Sign Up Page

수업 시간에 만든 회원가입 Form 예제를 바탕으로 입력 페이지와 결과 페이지를 연결한 과제입니다.  
사용자가 회원 정보를 입력하면 브라우저에서 입력 조건을 확인하고, 정상 입력된 값은 `result.html`의 표에 표시됩니다.

## 실행 방법

1. 프로젝트 폴더에서 `signup.html`을 실행합니다.
2. 이름, 아이디, 비밀번호, 이메일 등 회원 정보를 입력합니다.
3. `회원가입` 버튼을 누르면 입력 조건을 확인한 뒤 `result.html`로 이동합니다.
4. 결과 페이지의 `다시 입력하기` 버튼을 누르면 입력 화면으로 돌아갑니다.

Live Server로 실행해도 되고, 파일을 직접 열어도 기본 기능은 확인할 수 있습니다.

## 과제 요구 조건 적용

- `text`, `password`, `email`, `tel`, `date` 입력 사용
- 성별은 `radio`, 관심 분야는 `checkbox` 사용
- 지역은 `select`, 자기소개는 `textarea` 사용
- `fieldset`과 `legend`로 입력 항목 구분
- 이름, 아이디, 비밀번호, 이메일, 약관 동의에 `required` 적용
- 아이디는 영문/숫자 4~12자 `pattern` 적용
- 비밀번호는 `minlength="8"` 적용
- 전화번호는 `010-0000-0000` 형식 검사
- 상단 헤더에 `position: fixed`와 `z-index` 적용
- `:focus`, `:hover`, Box Model, 반응형 CSS 적용
- GET 방식으로 `result.html`에 값 전달
- `URLSearchParams`로 값을 읽어 `table`에 출력
- 관심 분야는 `getAll("interest")`로 복수값 처리

## 수업 내용을 이용해 추가한 부분

결과표를 기본 회원 정보, 선택 정보, 동의 정보로 나누어 보기 쉽게 만들었습니다. 선택하지 않은 값은 별도 문구로 표시하고, 비밀번호는 그대로 보여 주지 않고 입력한 길이만 알 수 있도록 점으로 바꾸었습니다.

생년월일은 `validation.js`에서 오늘 날짜를 계산해 `max` 값으로 넣었습니다. 또한 관심 분야 checkbox의 `name`을 모두 `interest`로 맞추고, 결과 페이지에서 `getAll()`을 사용해 선택한 항목을 전부 표시했습니다.

## 수업 외에 추가한 기능

### 회원가입 도우미

회원가입 화면 오른쪽에 간단한 규칙형 도우미를 추가했습니다. 사용자가 아래 내용과 관련된 질문을 입력하면 정해 둔 답변을 보여 줍니다.

- 아이디
- 비밀번호
- 이메일
- 전화번호
- 필수 항목
- 관심 분야
- 약관 동의

외부 AI API나 서버는 사용하지 않았습니다. 질문에 특정 단어가 포함되어 있는지 확인한 뒤 알맞은 답변을 출력하는 방식입니다.

### UI 변경

- SKALA 로고 적용
- 흰색 배경에 빨간색과 주황색 포인트 사용
- 회원가입 완료 안내 추가
- 관심 분야를 AI, 데이터, 개발 분야 등 20개로 확장
- 작은 화면에서 Form과 도우미가 화면 밖으로 넘치지 않도록 반응형 처리

## 작동 흐름

```text
회원 정보 입력
→ required / pattern / minlength 검사
→ 정상 입력 시 result.html로 이동
→ URLSearchParams로 전달값 확인
→ 결과표에 값 출력
→ 다시 입력하기로 signup.html 이동
```

## 파일 구성

```text
01_Signup_Result_Page/
├── signup.html
├── result.html
├── signup.css
├── validation.js
├── result.js
├── signupHelper.js
└── media/
    └── skala-logo.jpg
```

| 파일 | 내용 |
|---|---|
| `signup.html` | 회원가입 입력 화면과 도우미 구조 |
| `result.html` | 입력 결과를 보여 주는 페이지 |
| `signup.css` | 공통 디자인과 반응형 스타일 |
| `validation.js` | 생년월일 최대 날짜 설정 |
| `result.js` | 전달값을 읽어 결과표에 출력 |
| `signupHelper.js` | 질문 키워드에 맞는 답변 출력 |

## 확인 항목

- 빈 필수값 제출 차단
- 아이디 4~12자 및 영문/숫자 형식 검사
- 비밀번호 8자 이상 검사
- 이메일과 전화번호 형식 검사
- 약관 미동의 시 제출 차단
- 관심 분야 복수 선택 결과 출력
- 오늘 이후 생년월일 선택 제한
- 다시 입력하기 버튼 작동
- 가입 도우미 빠른 질문과 직접 질문 작동

> 이 프로젝트는 수업 과제용으로 만든 화면이며, 실제 회원정보를 서버에 저장하지 않습니다.
