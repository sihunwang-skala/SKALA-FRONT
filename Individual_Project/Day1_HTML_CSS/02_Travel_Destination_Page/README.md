# Chicago Travel Guide

> 2025년 6월에 방문한 미국 시카고의 명소, 음식, 여행 팁과 영상을 정리한 여행 소개 페이지입니다.  
> 수업에서 배운 HTML 시맨틱 태그와 CSS 레이아웃을 기본으로 사용하고, 개인 여행 사진과 Google My Maps, 음식별 맛집 순위 페이지를 추가했습니다.

## 과제 요구사항 확인

과제 안내에 나온 항목을 기준으로 최종 코드에 적용했는지 확인했습니다.

| 구분 | 요구사항 | 적용 내용 | 확인 |
|---|---|---|:---:|
| 시맨틱 구조 | `header`, `nav`, `main`, `section`, `article`, `aside`, `footer` | 각 영역의 역할에 맞게 태그를 나누고 `main`은 한 번만 사용 | ✅ |
| 고정 메뉴 | `position: fixed`, `z-index`, 내부 링크 | 스크롤 중에도 메뉴가 보이며 각 `section`의 `id`로 이동 | ✅ |
| Hero 영역 | 배경 이미지와 그라데이션 | 시카고 야경 사진 위에 어두운 그라데이션과 제목 배치 | ✅ |
| 명소 갤러리 | `figure`, `figcaption` 3개 이상 | 개인 여행 사진을 사용한 명소 카드 3개 구성 | ✅ |
| 이미지 설명 | 의미 있는 `alt` 작성 | 장소명과 사진 내용을 알 수 있도록 대체 설명 입력 | ✅ |
| 카드 효과 | `:hover`, `transition`, `transform` | 카드 이동과 이미지 확대 효과 적용 | ✅ |
| 먹거리·여행 팁 | `article`, `aside`, 2단 레이아웃 | 먹거리와 여행 팁을 역할에 맞게 구분해 나란히 배치 | ✅ |
| 영상 | `video`, `controls`, `poster`, `source` | 대표 이미지가 보이는 MP4 여행 영상 구성 | ✅ |
| 반응형 화면 | 미디어쿼리 | 화면 너비에 따라 카드 열 수와 콘텐츠 배치 변경 | ✅ |
| 출처 표시 | 외부 자료 출처 작성 | 음식 이미지와 맛집 순위 자료의 원본 링크 표시 | ✅ |

## 수업 내용을 바탕으로 확장한 부분

### 명소 카드와 고정 내비게이션

수업에서 배운 이미지 카드와 Grid 배치를 이용해 The Bean, Navy Pier, 3 Arts Club Cafe를 3개의 카드로 구성했습니다.  
각 카드는 `figure`와 `figcaption`으로 작성하고, 장소명과 설명, 촬영 시기를 함께 표시했습니다. 예제 이미지 대신 직접 촬영한 여행 사진을 사용했습니다.

상단 메뉴에는 `position: fixed`와 `z-index`를 적용했습니다. 스크롤을 내려도 메뉴가 화면 위에 유지되며, 메뉴를 누르면 여행 소개, 명소 갤러리, 지도, 먹거리, 여행 팁, 영상 영역으로 이동합니다.

### Flex & Grid와 반응형 화면

명소 갤러리와 맛집 순위 카드는 Grid로 배치했습니다. 먹거리 본문과 여행 팁, 영상과 영상 설명은 화면이 넓을 때 나란히 보이도록 구성했습니다.

화면이 좁아지면 미디어쿼리에서 카드의 열 수를 줄이고, 나란히 있던 영역을 위아래로 배치해 내용이 겹치지 않도록 했습니다.

### hover & transition 효과

명소 카드와 음식 카드에 마우스를 올리면 카드가 조금 위로 이동하고 이미지가 확대됩니다. 버튼과 링크에도 hover 효과를 적용해 색상이 자연스럽게 바뀌도록 했습니다.

### 여행 영상

여행 영상은 `video` 태그에 `controls`, `poster`, `source`를 사용했습니다. 영상이 재생되기 전에는 대표 이미지가 보이고, 재생·정지·음량 조절은 브라우저 기본 버튼으로 사용할 수 있습니다.

원본 MOV 파일은 일부 Chrome 환경에서 재생되지 않아 MP4 형식으로 변환한 파일을 사용했습니다.

## 더 나아간 부분

### Google My Maps 여행 지도

Cloud Gate, Navy Pier, 3 Arts Club Cafe at RH Chicago, Chicago Riverwalk의 위치를 직접 등록한 Google My Maps를 페이지에 넣었습니다.

별도의 API Key나 결제 설정 없이 `iframe` 방식으로 삽입했습니다. 명소 사진을 본 다음 위치를 바로 확인할 수 있도록 명소 갤러리 아래에 배치했습니다.

### 음식별 맛집 순위 페이지

`Food_Place_Ranking/index.html`을 별도로 만들어 시카고 대표 음식별 맛집을 정리했습니다.

- Deep-Dish Pizza: Chowhound 참고
- Chicago-Style Hot Dog: Time Out Chicago 참고
- Italian Beef Sandwich: The Infatuation 참고

각 음식마다 3곳씩, 총 9개의 식당을 카드로 구성했습니다. 1위, 2위, 3위는 금색, 은색, 동색 선으로 구분했습니다.

### Google 지도 연결

맛집 카드의 사진을 누르면 해당 식당 이름으로 Google 지도 검색 결과가 새 탭에서 열립니다. 사진에 마우스를 올리면 **Google 지도에서 보기** 안내가 표시됩니다.

맛집 순위 페이지 위와 아래에는 메인 여행 페이지의 먹거리 영역으로 돌아가는 링크를 넣었습니다.

## 작동 방식

1. `html/myTrip.html`을 브라우저에서 실행합니다.
2. 상단 메뉴를 누르면 여행 소개, 명소 갤러리, 지도, 먹거리, 여행 팁, 영상 영역으로 이동합니다.
3. 명소 카드와 음식 카드에 마우스를 올리면 hover 효과가 적용됩니다.
4. Google My Maps에서 여행지의 위치를 확인합니다.
5. 먹거리 영역의 **음식별 맛집 순위 보러가기** 버튼을 누르면 `Food_Place_Ranking/index.html`로 이동합니다.
6. 맛집 카드의 사진을 누르면 해당 식당의 Google 지도 검색 결과가 새 탭에서 열립니다.
7. 돌아가기 링크를 누르면 메인 페이지의 먹거리 영역으로 돌아옵니다.
8. 푸터의 **맨 위로** 링크를 누르면 페이지 최상단으로 이동합니다.

## 참고 사항

- 일부 맛집 이미지는 해당 식당의 실제 사진이 아니며, 해당 카드 아래에 따로 표시했습니다.
- 외부 이미지와 순위 자료는 각 페이지에 출처 링크를 작성했습니다.
- 원본 MOV 영상은 보관하고, 브라우저 재생을 위해 MP4 파일을 사용했습니다.
- 이 프로젝트는 HTML과 CSS를 중심으로 만든 수업 과제이며 별도의 서버나 데이터베이스를 사용하지 않습니다.
