# ANC HEADPHONE SHOP

> 수업 시간에 실습한 CSS Grid, Flexbox와 미디어 쿼리를 바탕으로 만든 반응형 상품 카드 갤러리입니다.  
> 프리미엄 ANC 헤드폰 6종의 가격, 출시 시기, 무게, 배터리와 주요 기술을 같은 형식으로 비교해 볼 수 있습니다.

## 주요 기능

- 실제 ANC 헤드폰 6종을 카드 형태로 표시
- CSS Grid를 이용한 기본 3열 배치
- 900px 이하 2열, 600px 이하 1열 반응형 화면
- Flexbox를 이용한 상단바와 카드 내부 정렬
- 카드 hover 이동, 그림자와 이미지 확대 효과
- 히어로 제목과 설명의 순차 등장 애니메이션
- 밝은 화면과 다크모드 전환
- 제품 사진, 제품명, 버튼에서 제조사 공식 페이지 연결
- 기술 용어 클릭 시 용어 안내로 이동
- ANC 작동 원리와 음파 다이어그램
- 제품 정보와 이미지 출처 정리
- WebLLM 기반 로컬 ANC AI Agent
- 페이지의 상품 정보와 기술 용어를 바탕으로 질문 응답
- AI 답변과 함께 상품 카드, 용어 설명, 출처 영역으로 이동

## 수업 내용을 바탕으로 확장한 부분

### 실제 상품 정보 적용

예시 상품명을 그대로 사용하지 않고 Sony, Bose, Sennheiser, JBL, Apple, Bowers & Wilkins의 헤드폰을 넣었습니다. 각 카드의 정보 순서를 맞춰 제품끼리 비교하기 쉽게 구성했습니다.

### 공식 사이트 연결

제품 사진, 제품명과 `공식 제품 보기` 버튼을 제조사 공식 페이지에 연결했습니다. 링크는 새 탭에서 열리며 `noopener noreferrer`를 함께 사용했습니다.

### ANC 기술 노트

ANC가 주변 소음을 줄이는 과정을 다음 네 단계로 정리했습니다.

1. 마이크가 주변 소음을 감지
2. 프로세서가 소리의 파형을 분석
3. 반대 방향의 음파 생성
4. 두 파형이 겹치면서 소음 감소

원래 소음과 반대 음파의 관계를 이해하기 쉽도록 SVG 음파 그림도 함께 만들었습니다.

### 용어 안내

상품 설명에 나오는 QN3, CustomTune, Smart Tx, Spatial Audio 등의 용어를 별도 영역에서 설명했습니다. 파란색 용어나 기술 칩을 누르면 해당 설명으로 이동하고 잠시 강조됩니다.

### 화면 구성

남색과 보라색을 기본 색상으로 사용하고, 실제 오디오 제품 소개 페이지처럼 보이도록 히어로 영역과 카드 디자인을 정리했습니다. 다크모드에서는 제목, 본문, 보조 정보와 강조값의 색을 나눠 가독성을 유지했습니다.

## 추가 구현 기능

### WebLLM 기반 로컬 ANC AI Agent

오른쪽 아래의 `ANC AI` 버튼을 누르면 채팅 패널이 열립니다. 사용자가 모델 시작 버튼을 눌렀을 때 WebLLM 모델을 준비하며, 모델 연산은 Web Worker에서 실행합니다.

상품과 용어 자료는 `data/headphones.json`에서 불러옵니다. 모델은 등록된 자료를 기준으로 답하고, 허용된 경우 제품 카드나 용어 설명으로 이동하는 버튼을 보여 주도록 구성했습니다.

이 기능은 브라우저에서 직접 모델을 실행하는 로컬 AI Agent이며, WebGPU 지원과 모델 실행 환경의 영향을 받을 수 있습니다. 모델 준비와 채팅 응답은 최신 Chrome의 Live Server 환경에서 별도로 확인해야 합니다. AI 기능이 작동하지 않아도 상품 카드, 다크모드, 용어 이동 등 과제의 기본 기능은 그대로 사용할 수 있습니다.

## 작동 방식

1. `index.html`을 열면 상품 카드가 화면 너비에 맞춰 배치됩니다.
2. 상단의 다크 버튼을 누르면 `script.js`가 `body.dark` 클래스를 전환합니다.
3. 상품 사진, 제품명이나 버튼을 누르면 제조사 공식 페이지가 새 탭에서 열립니다.
4. 파란색 기술 용어를 누르면 해당 용어 카드로 이동합니다.
5. 이동한 용어에는 강조 클래스가 잠시 적용됩니다.
6. ANC AI를 시작하면 `headphones.json`을 읽고 로컬 모델을 준비합니다.
7. AI가 반환한 허용 동작만 JavaScript가 확인한 뒤 실행합니다.

│   ├── bose-qc-ultra-2.png

    └── ancAgentWorker.js

| 파일 | 역할 |
|---|---|
| `index.html` | 상품 카드, ANC 기술 노트, 용어 안내, 출처와 AI 채팅 구조 |

| `data/headphones.json` | 상품 6개, 기술 용어 13개와 ANC 설명 자료 |
| `script/ancAgent.js` | 로컬 AI 모델 준비, 채팅과 허용 동작 처리 |
| `script/ancAgentWorker.js` | AI 모델을 별도 Worker에서 실행 |
| `images/` | 헤드폰 상품 이미지 보관 |
| `CHECKLIST.txt` | 제출 전 기능 확인 항목 |
| `CODE_CLEANUP_NOTE.txt` | 코드 정리 내용 기록 |

## 확인한 내용

- 큰 화면에서 상품 카드가 3열로 표시되는지 확인
- 900px 이하에서 2열로 변경되는지 확인
- 600px 이하에서 1열이 되고 상단 메뉴가 숨겨지는지 확인
- 다크 버튼을 눌렀을 때 화면 색상이 함께 변경되는지 확인
- 카드 hover 시 카드 이동과 그림자 효과가 적용되는지 확인
- 새로고침 시 히어로 제목과 설명이 순서대로 등장하는지 확인
- 상품 링크가 제조사 공식 페이지로 연결되는지 확인
- 용어 링크를 눌렀을 때 해당 설명으로 이동하고 강조되는지 확인
- `prefers-reduced-motion` 환경에서 움직임이 줄어드는지 확인
- ANC AI Agent는 WebGPU 지원 Chrome에서 모델 준비, 질문 응답과 페이지 이동을 별도로 확인

## 참고 사항

제품 정보와 이미지는 페이지 하단의 출처 영역에 정리했습니다.  
이미지와 제품 정보의 권리는 각 제조사와 원문 출처에 있습니다. 이 프로젝트는 수업 과제로 제작한 비상업적 결과물입니다.

ANC AI Agent는 외부 API 키를 사용하지 않고 브라우저에서 모델을 실행하도록 구성했습니다. 첫 실행 시 모델 파일을 준비하는 시간이 필요할 수 있습니다.
