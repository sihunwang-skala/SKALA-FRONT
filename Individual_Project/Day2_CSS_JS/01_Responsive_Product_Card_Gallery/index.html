TECH NOTE

소음을 소리로 줄이는 기술

Active Noise Cancellation

ABSTRACT

ANC는 마이크와 프로세서, 헤드폰 드라이버를 이용해 주변 소음과 반대되는 음파를 생성하는 능동형 소음 저감 기술입니다.

KEYWORDS

ANC / Anti-noise / Microphone / Processor / Destructive Interference

  1. 🎙️

    소음 감지

    외부 마이크가 주변의 소음을 전기 신호로 수집합니다.

  2. 🧠

    파형 분석

    프로세서가 입력된 소리의 크기와 주파수, 위상을 분석합니다.

  3. 🔊

    반대 음파 생성

    감지한 소음과 반대되는 위상의 음파를 헤드폰 드라이버로 출력합니다.

  4. 🎧

    소음 감소

    원래 소음과 반대 음파가 겹치면서 귀에 도달하는 소음이 줄어듭니다.

세 파형으로 보는 상쇄 간섭 (직접 제작한 다이어그램)

  • 원래 소음 n(t)
  • 반대 음파 a(t) ≈ -n(t)
  • 결과 n(t) + a(t) ≈ 0

ANC

마이크와 프로세서가 반대 음파를 만들어 소음을 능동적으로 줄이는 방식

PNC

이어패드와 이어팁이 귀를 물리적으로 막아 소음을 차단하는 방식

LIMITATIONS

ANC는 기차, 비행기 엔진, 에어컨처럼 비교적 일정하게 이어지는 저주파 소음에 효과적입니다.

사람의 말소리와 갑작스럽게 발생하는 높은 주파수의 소리는 상대적으로 줄이기 어렵습니다.

제품의 착용 상태, 이어패드 밀폐 정도, 주변 환경에 따라 실제 소음 감소 효과가 달라질 수 있습니다.

TERMS

용어 안내

파란색 링크나 기술 버튼을 누르면 필요한 용어로 바로 이동합니다.

ANC

Active Noise Cancellation의 약자다.

헤드폰의 마이크가 주변 소음을 감지하고 소음을 줄이는 신호를 만들어 외부 소리를 낮추는 기술이다.

MIC

Microphone의 줄임말이다.

주변 소음 감지, ANC 처리, 통화 음성 수집 등에 사용된다.

제조사마다 마이크를 세는 기준이 달라 숫자만으로 성능을 비교할 수 없다.

QN3

Sony가 WH-1000XM6에 적용한 HD 노이즈 캔슬링 프로세서다.

12개 마이크의 정보를 실시간으로 처리한다.

공식 출처: Sony 개발자 인터뷰 페이지
