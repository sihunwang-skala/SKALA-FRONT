# SKALA-FRONT

> Full-Stack Engineering 과정에서 진행한 HTML, CSS, JavaScript 개인 프로젝트입니다.
> 수업에서 학습한 내용을 바탕으로 기본 기능을 구현한 뒤, 프로젝트별로 추가 기능을 직접 설계하여 완성했습니다.

---

## 프로젝트 구성

```
Individual_Project
│
├── Day1_HTML_CSS
│   ├── 01_Signup_Result_Page
│   └── 02_Travel_Destination_Page
│
└── Day2_CSS_JS
    ├── 01_Responsive_Product_Card_Gallery
    └── 02_Interactive_Todo_App
```

# Day 1. HTML · CSS

## 1. Signup Result Page

회원가입 화면과 결과 화면을 구현한 프로젝트입니다.

### 주요 기능

- HTML Form
- 입력 유효성 검사
- GET 방식 데이터 전달
- 결과 페이지 출력
- 회원가입 도우미
- 반응형 레이아웃

### 폴더 구성

```
01_Signup_Result_Page
│
├── demo          프로젝트 실행 화면
├── media         사용한 이미지
├── signup.html
├── signup.css
├── validation.js
├── signupHelper.js
├── result.html
├── result.js
└── README.md
```

# Day 2. CSS · JavaScript

## 1. Responsive Product Card Gallery

CSS Grid와 Flexbox를 활용한 반응형 상품 카드 프로젝트입니다.

### 주요 기능

- Grid Layout
- Flexbox
- Hover Animation
- Dark Mode
- 기술 용어 이동
- ANC 기술 노트
- WebLLM 기반 ANC AI Agent

### 폴더 구성

```
01_Responsive_Product_Card_Gallery
│
├── data
├── demo
├── images
├── script
├── index.html
├── style.css
├── script.js
└── README.md
```

# 실행 방법

각 프로젝트 폴더의 README를 참고하거나 `index.html` 또는 `signup.html`을 Live Server로 실행합니다.

---

# 프로젝트 특징

- 수업 실습을 기반으로 프로젝트 확장
- HTML → CSS → JavaScript 순서로 난이도 상승
- 모든 프로젝트에 README 작성
- Demo 폴더에 실행 화면 정리
- Media 폴더에 사용한 리소스 정리
- 과제 요구사항을 모두 반영한 뒤 추가 기능 구현
