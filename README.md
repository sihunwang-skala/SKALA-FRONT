# SKALA-FRONT

SKALA 프론트엔드 수업에서 진행한 개인 과제를 모아둔 저장소입니다. HTML, CSS, JavaScript를 하루 단위 주제로 나눠 실습하고, 각 과제마다 요구사항을 확인한 뒤 조금씩 기능을 더 붙여봤습니다.

각 프로젝트의 자세한 실행 방법과 구현 내용은 폴더별 README에 정리했습니다.
